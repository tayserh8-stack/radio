# نظام إدارة قسم الرواتب - المواصفات التقنية

## نظرة عامة
تم تطوير نظام إدارة قسم الرواتب المتكامل كتطبيق ويب احترافي باستخدام React.js، يغطي كافة الجوانب التشغيلية والرقابية لإدارة الرواتب في مؤسسة متوسطة إلى كبيرة الحجم.

## المكونات التقنية

### التكنولوجيا المستخدمة
- **Frontend**: React.js 18.2.0
- **Routing**: React Router DOM 6.21.1
- **Icons**: Font Awesome
- **Build Tool**: Vite 5.0.8
- **State Management**: React Hooks (useState, useEffect, useContext)

### هيكل المشروع
```
src/features/payroll/
├── pages/                 # الصفحات الرئيسية
│   ├── PayrollManagement.jsx    # الصفحة الرئيسية
│   ├── PayrollDashboard.jsx     # لوحة القيادة
│   ├── PayrollWorkflow.jsx      # سير العمل
│   ├── PayrollPolicies.jsx      # السياسات والإجراءات
│   ├── PayrollAudit.jsx         # الرقابة والتدقيق
│   ├── PayrollIntegration.jsx   # التكامل التقني
│   ├── PayrollReports.jsx       # التقارير والتحليلات
│   └── PayrollProcessing.jsx    # معالجة الرواتب
├── components/            # المكونات القابلة لإعادة الاستخدام
├── hooks/                 # الـ hooks المخصصة
├── utils/                 # الأدوات والمساعدات
└── services/              # خدمات API
```

## الوحدات الوظيفية

### 1. الهيكل التنظيمي والوظائف
- تحديد الأدوار: مدير الرواتب، أخصائي رواتب، معالج رواتب، مدقق رواتب
- مصفوفة المساءلة (Accountability Matrix)
- مستويات السلطة والصلاحيات

### 2. دورة العمل (Workflow)
- 10 خطوات من تسجيل الحضور إلى الإيداع البنكي
- تتبع حالة كل خطوة (مكتمل، قيد التنفيذ، معلق)
- تحديد المواعيد النهائية الصارمة
- إشعارات وتنبيهات للمواعيد القريبة

### 3. السياسات والإجراءات (SOP)
- **حساب الاستقطاعات**:
  - ضريبة الدخل (15%)
  - التأمين الاجتماعي (9% - موظف + صاحب عمل)
  - التأمين الصحي (3%)
  - الصندوق التقاعدي (5%)

- **البدلات**:
  - بدل سكن: 25% من الراتب الأساسي (بحد أقصى)
  - بدل نقل: حسب المسافة
  - بدل غذاء واتصال: مبالغ ثابتة

- **المواعيد النهائية**:
  - يوم 26: إغلاق الحضور
  - يوم 27-28: معالجة المتغيرات
  - يوم 29: الحساب والخصومات
  - يوم 30: الموافقة والتدقيق

### 4. الرقابة والتدقيق (Internal Controls)
- **فصل المهام** (Segregation of Duties)
- **المطابقة البنكية** (Bank Reconciliation)
- **سجل التدقيق** (Audit Trail)
- **مؤشرات الأداء** (KPIs)
- ضوابط الوصول والأمان

### 5. التكامل التقني (System Integration)
- **أنظمة HRIS**: إدخال البيانات الأساسية للموظفين
- **أنظمة ERP**: القيود المحاسبية
- **البنوك**: ملفات التحويل الإلكتروني
- **الهيئات الحكومية**: الإقرارات الضريبية
- **الأمان**: تشفير AES-256، SSL، جدران الحماية

### 6. التقارير والتحليلات (Reporting & Analytics)
- **التقارير الشهرية**: ملخص الرواتب، الاستقطاعات
- **تحليل التكاليف**: حسب الأقسام، التكاليف الإجمالية
- **انحراف الميزانية**: مقارنة المخطط بالفعل
- **تدفق النقد**: توقعات المدفوعات المستقبلية
- **تحليل العمل**: توزيع التكاليف حسب الفئات
